How To Use

Step .1: Download Wayback Launcher 

Step .2: Extract The Zip File

Step .3: Open Wayback Launcher.exe

Step .4: Press The Settings Icon

Step .5: Sign In

Step .6: Select Your Game Build Folder

Step .7: Press Home 

Step .8: Press Launch
